# Habit & Productivity Tracker

A productivity management web application built using React.js and Firebase.

## Features

- User Authentication
- Habit Tracking
- Streak Monitoring
- Achievement Badges
- Reminder System
- Calendar Tracking
- Analytics Dashboard
- Browser Notifications
- PDF Export

## Tech Stack

- React.js
- Firebase Authentication
- Firestore
- Chart.js
- jsPDF
- CSS
- Vercel
